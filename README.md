# sori-kao

> 한국어 짧은 표현의 **음운 특징**을 분석해, 입력이 주는 **소리 인상**을 설명하는 규칙 기반 Python 프로젝트.
> 카모지는 핵심 목표가 아니라, 해석 결과를 보여주기 위한 선택적 부가 출력이다.

## 무엇을 하는가

`아싸`, `우우`, `빠샤`, `하...`, `두근두근`처럼 의성어·의태어·감탄사·구호에 가까운 짧은 한국어 표현을 한글 음절 단위로 분해하고, 밝은/어두운 모음, 된소리·거센소리, 받침, 반복, 문장부호 같은 특징을 수치화한다. 그 결과를 논문 기반 음운 인상 카테고리와 비교해 “이 표현이 어떤 소리 느낌을 강하게 갖는지”를 설명한다.

```text
입력:  아싸아싸!!
해석:  밝음·가벼움, 반복·리듬, 강함·격렬함
이유:
- 밝은 모음 비율이 높아 밝고 가벼운 인상 점수가 올라갔습니다.
- 같은 음절 패턴이 반복되어 반복·리듬 인상이 강하게 감지되었습니다.
- 느낌표가 있어 강도 신호가 보조적으로 반영되었습니다.
```

## 핵심 방향

- **해석이 먼저**: 목표는 카모지를 추천하는 것이 아니라, 입력의 음운 특징이 어떤 인상으로 해석되는지 설명하는 것이다.
- **논문 기반 카테고리**: 대화 상황이나 감정 이름을 핵심 분류로 쓰지 않는다. 카테고리는 한국어 의성어·의태어 연구에서 다루는 반복, 교체, 모음조화, 밝음/어두움, 강함/부드러움 같은 음운 인상 축을 기반으로 둔다.
- **규칙 기반 + 설명 가능**: 생성형 AI/LLM/외부 API를 쓰지 않고, 사람이 검토할 수 있는 특징 계산과 NumPy 점수 모델로 동작한다.
- **경향으로만 해석**: 음운상징은 절대 규칙이 아니므로, 결과는 감정 정답이 아니라 해석 후보와 점수 근거로 제시한다.
- **카모지는 후순위**: 필요하면 해석 결과에 어울리는 카모지를 나중에 붙일 수 있지만, 프로젝트의 평가 기준은 카모지가 아니라 음운 해석의 근거와 일관성이다.

## 논문 기반 해석 카테고리

| 해석 카테고리 | 주요 근거 |
|---|---|
| `밝음·가벼움` | 양성/밝은 모음 계열, 모음조화 |
| `어두움·무거움` | 음성/어두운 모음 계열, 무거운 인상 |
| `작음·섬세함` | 밝은 모음이 만드는 작고 가벼운 인상 |
| `큼·둔중함` | 어두운 모음이 만드는 크고 무거운 인상 |
| `강함·격렬함` | 된소리, 거센소리, 강한 자음 인상 |
| `부드러움·흐름` | 비음/유음, 완만하고 이어지는 인상 |
| `반복·리듬` | 완전 반복, 부분 반복, 반복 의태어 구조 |
| `지속·여운` | 반복 지속, 말줄임, 종성 울림 |
| `변화·교체감` | 자음 교체, 모음 교체, 교체 반복(확장 축, MVP 미산출) |
| `막힘·단절감` | 받침 밀도, 어말 자음, 끊기는 느낌 |

## 처리 흐름

```text
입력
→ 전처리
→ 한글 음절 분해
→ 음운 특징 추출
→ 14차원 feature vector 생성
→ 논문 기반 해석 카테고리 점수 계산
→ Top-3 해석 후보 산출
→ 해석 이유와 신뢰도 출력
```

## 한계

- 음운 특징과 의미 인상은 절대 대응하지 않는다. 결과는 정답 판정이 아니라 경향 해석이다.
- 현재 카테고리 점수의 실제 숫자는 논문에서 가져온 값이 아니라, 문헌 기반 특징을 구현하기 위한 초기 휴리스틱이다.
- 일반 문장은 의성어·의태어보다 음운상징성이 약할 수 있다. 따라서 이 프로젝트는 짧은 표현을 주 대상으로 하고, 일반 문장은 낮은 신뢰도로 보수적으로 다룬다.
- `하암`처럼 소리만으로 실제 상황을 단정하기 어려운 표현은 뜻을 확정하지 않는다. `ㅋㅋ`, `ㅠㅠ` 같은 자모 반복은 표기 보조 신호로만 구분하고, 핵심 해석과 분리한다.
